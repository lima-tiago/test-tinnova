## Instalando as dependências do Projeto

### `yarn` ou `npm install`

Este comando irá instalar todas as dependências necessárias para a aplicação rodar
<br>

## Executando a aplicação

### `yarn dev` ou `npm run dev`

Executa o aplicativo no modo de desenvolvimento.\
Abre o [http://localhost:3000](http://localhost:3000) para visualizá-lo no navegador.

A página será recarregada se você fizer edições.\
<br>

## Documentação

Para verificar a documentação dos principais components basta executar o comando `yarn storybook`

## Testes

Para verificar os testes da aplicação basta executar o comando `yarn test`
